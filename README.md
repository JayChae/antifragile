# 안티프래질 (Antifragile) — 독서 노트

나심 니콜라스 탈레브의 *안티프래질*을 읽으며 정리한 노트입니다.

https://jaychae.github.io/antifragile/

## 목차

- 서문
- 1장, 2장: 히드라 - 과잉반응 - 과잉보상
- 3장: 고양이와 세탁기
- 4장: 안티프래질 시스템 속 프래질한 개인
- 5장: 무작위성이 장기적으로 안정적이다
- 6장: 무작위성을 억제하지 말고 투입하라
- 7장: 어설프게 개입하지 마라
- 8장, 9장: 예측하지 마라
- 10장: 세네카 훈련
- 11장: 록스타와 결혼하지 마라
- 12장: 탈레스의 달콤한 포도
- 13~17장: 대학은 번영의 원인이 아니라 결과다
- 18장: 비선형
- 19~22장: 프래질을 제거
- 23장: 안티프래질 윤리
